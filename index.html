Membangun BATIM GADAI Mobile

Aplikasi Android yang menjadi antarmuka utama bagi nasabah PT Bintang Timur dalam mengakses layanan gadai secara digital dan memantau status pinjaman.

Peran
Mobile Developer
Platform
Android (Flutter)
Status
Sedang Dikembangkan

01 Arsitektur Sistem

Aplikasi ini terhubung langsung dengan sistem web BATIM GADAI melalui REST API berbasis Laravel Sanctum. Platform digital ini terbagi menjadi dua komponen utama: Web System untuk Superadmin/Officer (repo terpisah) dan Mobile App ini khusus untuk nasabah.

02 Keunggulan & Fitur Utama

Login Tanpa Password (OTP)
Nasabah cukup input nomor handphone, terima OTP via WhatsApp, dan buat PIN 6 digit untuk keamanan akses.
Pembayaran Online (Midtrans)
Perpanjangan masa gadai dan pelunasan dapat dilakukan secara online melalui integrasi Payment Gateway Midtrans kapan saja tanpa perlu ke outlet.
Pantau & Simulasi Real-time
Memantau seluruh transaksi gadai aktif, tanggal jatuh tempo, riwayat, serta fitur simulasi untuk estimasi nilai pinjaman berdasarkan harga pasar.

03 Manajemen Hak Akses

Sistem membedakan dua tipe pengguna: Pengunjung (login via OTP tapi belum verifikasi CIF) yang hanya bisa mengakses Beranda dan Simulasi, serta Nasabah (terverifikasi KTP + CIF) yang memiliki akses penuh ke fitur Pinjaman, Pembayaran, dan SBG.

04 Tech Stack & Desain

Flutter 3.x Riverpod 2.x Dio 5.x Go_router 13.x Midtrans SDK Geolocator

Membangun Fo-Learn
dari Nol

Aplikasi edukasi berbasis Flutter yang dirancang untuk menarik perhatian Gen-Z melalui pembelajaran format pendek, simulasi ujian real-time, dan alur langganan premium yang mulus.

Peran
Solo Dev & UI/UX
Platform
Android (Flutter)
Status
Selesai (Shipped)

01 Masalah

Siswa SMA yang bersiap menghadapi UTBK sering merasa bosan dengan aplikasi bimbel konvensional yang kaku dan penuh teks panjang. Hal ini bertolak belakang dengan kebiasaan konsumsi konten Gen-Z yang terbiasa dengan video pendek bertempo cepat.

02 Solusi: Flash Learning

Membawa materi UTBK dalam format Flash Learning layaknya TikTok/Reels. Setiap sesi dikemas di bawah 90 detik, membuat belajar terasa ringan dan adiktif.

Fitur ini dipadukan dengan TryOut Simulator berbatas waktu, penilaian otomatis, dan analitik performa. Sistem Freemium diatur dengan integrasi Midtrans Payment Gateway untuk akses materi premium.

03 Tantangan Teknis

Manajemen State Video & Memory Leak
Masalah: Video yang di-scroll tetap berjalan di latar belakang, menyebabkan lag.
Solusi: Mengimplementasikan lifecycle manager khusus di Flutter sehingga controller dimatikan saat di luar layar, menjaga performa tetap 60fps.
Webhook Pembayaran Real-Time
Masalah: Membuka akses premium otomatis saat pembayaran sukses tanpa refresh.
Solusi: Firebase Cloud Function menerima webhook Midtrans dan mengupdate Firestore. Aplikasi menggunakan StreamBuilder untuk mendeteksi perubahan seketika.
